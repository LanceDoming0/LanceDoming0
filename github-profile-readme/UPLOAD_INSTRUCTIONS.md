# How to Use This as Your GitHub Profile README

1. Create your GitHub account and choose a username.
2. Create a **public repository with exactly the same name as your GitHub username**.
3. Check **Add a README file**.
4. Replace the repository README with the included `README.md`.
5. Replace `ADD_YOUR_NETLIFY_URL_HERE` with your published portfolio address.
6. Pin your four strongest repositories on your GitHub profile.

Do not upload private credentials, client data, Discord tokens, `.env` files, or database passwords.
