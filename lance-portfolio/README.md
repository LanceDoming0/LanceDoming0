# Lance — Developer & IT Support Portfolio

A responsive portfolio website for **Lance Albert D. Aganon**, focused on junior web development, IT support, digital assistance, Discord bot development, and gaming-community projects.

## Highlights

- Semantic HTML5 and responsive Bootstrap layout
- Dark interface with electric-blue accents
- Scroll-reveal transitions and mouse-following 3D card effects
- Project filters, image lightbox, animated counters, and rotating titles
- Experience and education timelines
- Netlify-ready contact form and downloadable resume
- Accessible reduced-motion and touch-device fallbacks

## Featured Work

- Community and business websites
- Discord multi-bot dashboard
- PC repair and IT support
- FiveM vehicle and livery projects
- SQL and reporting demonstration

## Run Locally

Open `index.html` in a browser, or use a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deploy

This static website can be deployed through Netlify by importing this repository. No build command is required; publish the repository root.

> The contact form uses Netlify Forms and will only submit correctly after deployment on Netlify.

## Privacy and Public Files

Only public portfolio material is included. No passwords, API keys, bot tokens, private databases, or client credentials are stored in this repository.
